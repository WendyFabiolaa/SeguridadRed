¿Puedes descifrar este mensaje?Descifre este [mensaje](https://artifacts.picoctf.net/c/160/cipher.txt) usando esta clave "CYLAB".

```
rgnoDVD{O0NU_WQ3_G1G3O3T3_A1AH3S_2951c89f}
```
